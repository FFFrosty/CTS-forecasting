**CTS 2026算法大赛来了**

**背景介绍**

PREFACE

**计算交通科学国际研讨会** （Computational Transportation Science，简称CTS）创立于2009年，旨在集中交通运输管理学科研究者的智慧，充分融合现代信息技术与管理科学、系统科学、统计学和运筹学的研究方法，着重研究交通运输系统资源的优化配置与运营过程的科学化管理，以保障交通运输系统的畅通高效、安全绿色运行；建立针对我国交通运输管理的集成学科建设、人才培养、科学研究、管理创新的综合平台，为我国交通运输管理事业的发展做出贡献。

第17届CTS会议将于2026年8月在长沙中南大学举办。为助推交通强国建设，满足交通运输数字化转型发展需要，加强交通运输领域高科技技术人才队伍建设，激发科研技术人才创新创造活力，促进交通运输管理领域的交流与合作，经研究决定举办第四届“CTS杯”交通科技竞赛。


**赛题简介**

**港口拖轮AIS数据预测**

拖轮是港口作业的核心力量，被称为"港口小巨人"。每一艘靠泊、离泊的万吨巨轮背后，都离不开拖轮的精准顶推与拖带。Automatic Identification System (AIS)（自动识别系统）数据以秒级频率记录着拖轮的位置、航速、航向，这些海量时空数据隐藏着港口运营的深层规律。本次竞赛提供某大型港口一个月的真实拖轮AIS数据，涵盖位置、航速、航向、转向率等完整运动学特征。要求参赛者基于所提供的拖轮AIS数据开展作业数量预测、进出流量预测、异常行为检测等任务。

1

**数据集**

本次竞赛基于中国北方某大型港口2018年1月1日至1月31日共31天的真实拖轮AIS数据，包含超过240万条秒级时空记录。数据集字段包括：船舶标识（MMSI）、经度（x）、纬度（y）、对地航向（COG）、真航向（True Heading）、对地航速（SOG）、转向率（ROT）和时间戳（精确到秒）。

数据覆盖118艘拖轮，日均记录约8万条，采样间隔中位约10秒。数据为真实场景采集，未经人工清洗，包含自然发生的传感器异常、信号缺失和航速离群点等问题。

数据划分为：

（1）训练集：2018年1月1日至1月24日（共24天），提供完整的AIS字段供选手建模训练；

（2）验证集：2018年1月25日至1月31日（共7天），仅提供验证集当天出现过的唯一船舶总数。

参赛者可基于训练集开展特征工程、模型训练和参数调优，最终提交预测结果。



**赛题任务**

赛题A：拖轮作业数量预测
【任务描述】给定历史AIS数据，预测未来某一时段、指定空间区域内处于“活跃作业状态”的拖轮数量。“活跃作业状态”定义为：拖轮对地航速（SOG）在2~10节范围内（即处于拖带作业或机动状态），且该拖轮在目标时间窗口内至少存在3条AIS记录（排除瞬时噪声）。空间区域划分如下：以该港口核心港区中心（经纬度坐标为117.79°E, 38.97°N）为圆心，按半径划分为三个圈层：核心区（0~3km）、近港区（3~10km）和外围区（10~30km）。选手也可自行采用网格法或聚类法定义空间区域。
【预测目标】以1小时为时间窗口，预测验证集期间（1月25日至31日）每个时间窗口、每个圈层内的活跃拖轮数量。提交文件格式为CSV，每行包含：时间窗口标识、圈层标识、预测拖轮数量。
赛题B：拖轮进出流量预测
【任务描述】根据相邻时间窗口内拖轮空间位置的变化，判断每艘拖轮是否跨越空间区域边界，统计各圈层之间的跨区迁移拖轮数量。边界穿越判定规则如下：若某拖轮在第t个时间窗口位于区域A，在第t+1个时间窗口位于区域B（A≠B），则判定该拖轮发生了跨区迁移。其中，从区域A迁移至区域B的拖轮总量计入”跨区迁移量”。
【预测目标】以1小时为时间窗口，输出验证集期间各圈层之间的迁移量（如核心区→近港区、近港区→核心区等）。提交文件格式为CSV，每行包含：时间窗口标识、源圈层、目标圈层、预测跨区迁移量。
